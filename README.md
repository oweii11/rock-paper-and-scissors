## 🧠 How It Works

1. Player selects **Rock**, **Paper**, or **Scissors**.
2. Computer randomly selects an option.
3. Winner is determined based on classic rules:
   - Rock beats Scissors
   - Scissors beats Paper
   - Paper beats Rock
4. Score is updated accordingly.
